# 📌 CSS (Cascading Style Sheets)
### 2025. 07. 00. 0요일

## 💡 CSS란?
- 스타일 시트 (Style Sheet) 언어이다.
- HTML 문서에 있는 요소들을 선택해 스타일을 적용할 수 있다.

## 선택자 (Selector)
- 어떤 태그를 선택할지 규정하는 문법

***

## 기본 선택자

__`*` 전역 선택자__
- 모든 태그를 선택한다

__Tag 유형 선택자__
- 해당 태그를 선택한다

__`#id` ID 선택자__
- id 요소를 선택한다

__`.class` class 선택자__
- 클래스 요소를 선택한다

## 자손 선택자와 후손 선택자

### 자손 선택자
`a > b` 선택자 a의 자손에 위치하는 선택자 b를 선택
`ex: div > span`

### 후손 선택자
`a b` 선택자 a의 후손(하위)에 위치하는 선택자 b를 선택한다.
