# 쿠키 | 로컬스토리지

## 쿠키 (Cookie)

### 실생활 비유

- 스탬프를 찍어줌, 자동 제시

<br>

> 서버가 브라우저에 정보를 심어주고, 이후 그 사이트에 접수할 때마다 브라우저가 자동으로 첨부해서 보냄

- 만료일 설정 가능 (ex: 30일 후 자동 삭제)
- 자동 전송이 핵심 - 개발자가 코드 짤 필요 x
- 세션 ID나 사용자 설정 값을 담는데 주로 사용
- 보안 옵션으로 `HttpOnly`(JS에서 접근 불가). `Secure`(HTTPS만) 설정가능

## 로컬스토리지 (localStorage)

### 실생활 비유

- 집 서랍에 물건 보관

<br>

> 브라우저에 영구적으로 저장되고, 직접 지우기 전에는 절대 안 사라짐

- 쿠키와 달리 서버에 자동 전송 안 됨 -> 개발자가 직접 꺼내서 붙어야 함
- 브라우저 닫아도 남아있음
- 용량이 쿠키보다 훨씬 큼 (약 5MB)
- 다크모드 설정, 최근 본 항목, 토큰 저장 등에 활용
